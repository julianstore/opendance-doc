## OptionDance REST API Documentation

**Specifications**

- [API Reference](/developer/lake/apis/overview): RESTful API reference.
- [Action Protocol](/developer/lake/action-protocol): a memo based JSON protocol to swap crypto, add and remove liquidity.

## API Endpoints

OptionDance provides a simple, developer-friendly APIs for 3rd-party developers.

| Name        | Endpoint                        |
| ----------- | ------------------------------- |
| Primary     | https://api.4swap.org/api       |
| Alternative | https://mtgswap-api.fox.one/api |

You can use any of them to construct your API requests in form:

```
HTTP_METHOD ${API_BASE}/${API_PATH}
```
